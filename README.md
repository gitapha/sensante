\# SenSante Assistant de pre-diagnostic medical pour le Senegal. #

\# Description SenSante utilise le Machine Learning pour aider au pre-diagnostic des maladies courantes (paludisme, grippe, typhoide) a partir des symptomes du patient. #

\# Structure du projet-

&#x20;`data/` : Donnees patients (CSV)-

&#x20;`models/` : Modele ML serialise-

&#x20;`api/` : API FastAPI-

&#x20;`frontend/` : Interface web-

&#x20;`notebooks/` : Scripts d'exploration #
